# Changelog

## 1.2.0 - 2026-04-26

- Added responsive layout primitives: AppShell, ContentContainer, PageHeader, SectionHeader, Stack, Cluster, ResponsiveGrid, Topbar, and Sidebar.
- Added layout guidelines for mobile, tablet, and desktop composition.
- Improved Input accessibility with generated IDs, label binding, error/helper descriptions, and invalid state wiring.
- Improved Tabs keyboard behavior with ArrowLeft, ArrowRight, Home, and End support.
- Hardened ProgressBar against invalid max values.
- Applied Checkbox and Toggle `className` to their root labels.
- Rewrote README and component inventory to remove generated artifacts and reflect current maturity.
- Updated package version to 1.2.0.

## 1.1.0 - 2026-04-26

- Added enterprise roadmap documentation.
- Added Wellness Connect brand guidelines.
- Added token architecture documentation.
- Added accessibility guidelines.
- Added semantic and component token aliases to the theme.
- Improved accessibility states for existing components.
